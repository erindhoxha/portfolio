I've built this website for Yoga with Kassandra to sell/rent videos of her Yoga channel.

GETTING STARTED

Client overview

Yoga With Kassandra is a Yoga channel where people can watch videos and do Yoga courses at home.

Their aim is to create accessible, easy to follow yoga classes that you can do in the comfort of your own home.

Research

The problem

Yoga With Kassandra had a website which users can view content, but the website wasn't meeting her design, branding and navigation of the website needs.

We collaborated with YWK to make a website which users can log in - Buy membership and then watch the courses.

Research

Solution

While the problem was how the users could navigate through, we've created a flowchart of how the users will navigate from Clases to the video.

The sketch overviews the transition from the main album e.g Classes, to the item they want to see.

Research

App and Web differences

Yoga With Kassandra has an App published for Android & iOS, so it meant that the website should be consistent same as the app.

Following the UX Rules, we've came up with the flowchart of how the thumbnails, buttons and the course details should look like.

DESIGN

Landing Page

The landing page was the Main focus where users first engage.

The key elements were to Sign up, and see the App directly while having the option to buy the access weekly or monthly.

For best user experience, we've added a phone which showcases a feature of the app, to engage with the customers more.
